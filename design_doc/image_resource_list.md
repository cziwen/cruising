# 图片资产清单

此文档列出休闲类像素风格（Stardew Valley 参照）游戏中所需的所有 PNG 资源。  
将此表作为独立的"图片需求清单"使用，后续在设计评审或美术实现阶段对照校验。

---

## 文档结构

本文档已按资源类型拆分为多个子文档，便于分类管理和查阅：

- **[场景背景](image_resources_backgrounds.md)** - 海洋背景、港口背景等场景图片
- **[船只](image_resources_ships.md)** - 玩家船只、敌方船只等船只图片
- **[UI元素](image_resources_ui.md)** - 按钮、面板、图标等界面元素
- **[商品图标](image_resources_goods.md)** - 交易界面使用的商品图标
- **[船员头像](image_resources_crew.md)** - 船员管理界面使用的头像
- **[天体元素](image_resources_celestial.md)** - 太阳、月亮等天体元素
- **[补充说明](image_resources_notes.md)** - 可选资源、优先级建议等

---

## 快速索引

### 1. 场景背景 (Backgrounds)
- IMG_BG_OCEAN - 海洋背景
- IMG_BG_PORT_1/2/3/N - 港口背景
- IMG_BG_SAILING - 航行场景

详见：[image_resources_backgrounds.md](image_resources_backgrounds.md)

### 2. 船只 (Ships)
- IMG_SHIP_PLAYER_0..6 - 玩家船只（0-6级进化）
- IMG_SHIP_ENEMY_0 - 敌方船只
- IMG_SHIP_SINKING - 沉船动画状态

详见：[image_resources_ships.md](image_resources_ships.md)

### 3. UI 元素 (UI Elements)
- IMG_UI_BTN - 按钮（默认/悬停/按下）
- IMG_UI_PANEL - 面板背景框
- IMG_UI_DIALOG_BG - 对话框背景
- IMG_UI_BAR_* - 进度条相关
- IMG_UI_ICON_* - 各种图标
- IMG_UI_MAIN_* - 主菜单相关

详见：[image_resources_ui.md](image_resources_ui.md)

### 4. 商品图标 (Goods Icons)
- IMG_GOODS_GOLD - 金币图标
- IMG_GOODS_FOOD - 食物图标
- IMG_GOODS_WOOD - 木材图标
- IMG_GOODS_SPICE - 香料图标
- IMG_GOODS_METAL - 金属图标

详见：[image_resources_goods.md](image_resources_goods.md)

### 5. 船员头像 (Crew Avatars)
- IMG_CREW_AVATAR_1/2/N - 船员头像
- IMG_CREW_PLACEHOLDER - 船员占位符

详见：[image_resources_crew.md](image_resources_crew.md)

### 6. 天体元素 (Celestial Bodies)
- IMG_CELESTIAL_SUN - 太阳
- IMG_CELESTIAL_MOON - 月亮

详见：[image_resources_celestial.md](image_resources_celestial.md)
