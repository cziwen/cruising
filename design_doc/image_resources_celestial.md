# 天体元素图片资源

## 概述

天体元素包括太阳和月亮，它们由 `CelestialLayer` 管理，根据昼夜系统的时间进度在屏幕上方移动。

---

## 资源列表

### 1. 太阳 (Sun)

| Asset ID | 路径 | 描述 |
| :--- | :--- | :--- |
| IMG_CELESTIAL_SUN | `assets/images/pixel-sun-icon.png` | 白天显示的像素太阳 |

- **尺寸**: 约 64x64 像素
- **显示时段**: 06:00 - 18:00 (进度 0.0 - 0.5)
- **技术说明**: 在日出/日落时有淡入淡出效果。

---

### 2. 月亮 (Moon)

| Asset ID | 路径 | 描述 |
| :--- | :--- | :--- |
| IMG_CELESTIAL_MOON | `assets/images/moon-in-pixel-art.png` | 夜晚显示的像素月亮 |

- **尺寸**: 约 64x64 像素
- **显示时段**: 18:00 - 06:00 (进度 0.5 - 1.0)
- **技术说明**: 在月出/月落时有淡入淡出效果。

---

## 运动轨迹

天体在屏幕上方沿弧形轨迹从右向左移动：
- **X 轴**: 线性移动。
- **Y 轴**: 正弦波形轨迹。
- **位置计算**: 详见 `2.8 day_night_system.md`。
